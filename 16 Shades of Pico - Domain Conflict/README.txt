- DOMAIN CONQUEST -
- A game by 16 Shades of Pico - 

Made in Porto Graphics / IEEE Retro Game Jam using PICO 8

Team:
    Catarina Vieira
    Francisco Múrias
    Luís Reis
    Nina Belllini

Description:
    In a psicadelic trip stretching the Pico Engine to its limits, this first person multiplayer
     split screen extravaganza pits two players "dis-COVER-ing" the floor in a trippy virual world
      to conquer the cyber grid before the time runs out!

Instructions:

    Keyboard:

    Start Z
    Toggle Sound X

    Player 1 - Arrow keys
    Player 2 - ESDF Keys

    Recommended use with xbox controller.
